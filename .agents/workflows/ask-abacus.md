description: Utiliser Abacus AI pour obtenir une deuxième opinion ou une analyse approfondie
---

Ce workflow permet à Antigravity d'interroger Abacus AI pour résoudre des problèmes complexes ou optimiser le code.

1. Définir la question ou la tâche complexe.
2. Exécuter la commande suivante :
// turbo
```powershell
node scripts/ask_abacus.js "Votre question ici"
```
3. Analyser la réponse d'Abacus et l'appliquer au code source si elle est pertinente.
